# 📅 LiveReserve PRD（プロダクト要求仕様書）

> **キャッチコピー**: 「配信予約、もうStudioを開かない。」 / 「配信枠作成を、ワンクリックで。」

---

## 1. 📋 概要

YouTube Live の配信予約を自動作成するためのツール。

配信者は毎回 YouTube Studio を開いて以下を設定する必要がある。

- タイトル
- 概要欄
- 公開設定
- 配信日時
- サムネイル
- OBS 用配信枠

本サービスはこれらをテンプレート化し、**ワンクリックで配信予約を作成**できるようにする。

---

## 2. 🎯 解決したい課題

現在の配信予約作成には以下の問題がある。

- 毎週同じ内容を入力している
- タイトル変更を忘れる
- 概要欄更新を忘れる
- 配信URL取得に手間がかかる
- 配信枠作成を忘れる

特に定期配信を行う VTuber や配信者は、同じ作業を何度も繰り返している。

## 4. 🚀 MVP

### 🔑 Googleログイン
YouTube アカウントでログインする。

### 📝 テンプレート作成

保存できる項目:

- タイトル
- 概要欄
- 公開範囲
- サムネイル

**例**

タイトル
```
【ゲーム名】リーグ第{{round}}節！今日も頑張る！【配信者名】
```

概要欄
```
いつもありがとうございます！

#ハッシュタグ1
#ハッシュタグ2
```

### ⚙️ 配信予約作成

**入力**
- 日時

**実行**
- YouTube 配信予約作成
- 配信ストリーム作成
- 自動紐付け

**出力**
- 配信URL
- videoId

### 📺 配信一覧

作成済み配信を表示。

表示項目:

- タイトル
- 日時
- URL
- 公開状態

---

## 5. 🔮 将来機能

### 🔁 定期配信

例: 毎週日曜21:00

ボタン一つで以下を実行できる。

- 来週分作成
- 来月分作成
- 3ヶ月分作成

### 🏷️ タイトル変数

利用可能変数:

- `{{date}}`
- `{{month}}`
- `{{day}}`
- `{{weekday}}`
- `{{round}}`

**例**
```
【ゲーム名】リーグ第{{round}}節【{{date}}】
↓
【ゲーム名】リーグ第12節【2026/06/07】
```

### 🖼️ サムネイル自動生成

テンプレート画像に対して、以下を自動合成する。

- 日付
- 節数
- タイトル

### 📣 配信SNS投稿

X 投稿文を自動生成する。

**例**
```
配信開始まであと1時間！

https://youtube.com/xxxxx
```

### 🔔 Discord通知

予約完了時に Discord へ通知する。

## 6. ⚙️ 非機能要件

### 🎨 UI
方針:

- シンプル
- モダン
- Apple風
- 配信者向け

不要な機能は置かない。

### ⚡ パフォーマンス
配信予約作成の目標: **3秒以内**

### 🛡️ 可用性
個人開発を前提とする。障害発生時はユーザーへエラー内容を表示する。
