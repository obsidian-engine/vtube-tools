## 個人的なカスタマイズ内容

Obsidian Engine テーマをベースにした、水色グラデーションのシンプルなコメント表示テンプレートです。

## 特徴

### デザイン
- **水色グラデーション**: 爽やかな水色系のグラデーション（#53B7FF → #81E3DD）
- **名前タグ**: グラデーション背景 + 白文字でユーザー名を強調
- **メッセージ**: 白背景 + ダークグレー文字で読みやすさを重視
- **スパチャ/メンバーシップ**: 全体をグラデーションで目立たせる
- **フォント**: Noto Sans JP で日本語に最適化

### レイアウト
- 縦並び（上から下へ流れる）シンプルな配置
- 角丸12pxで柔らかい印象
- 軽めのシャドウで立体感

## 個人的な使用メモ

### わんこめアプリでの設定
1. わんこめアプリを起動
2. テンプレート選択画面で「aqua-gradient」を選択
3. OBS Studioのブラウザソースとして表示

### OBS Studioでの設定
1. ソース追加 → ブラウザ
2. URLにわんこめのローカルサーバーアドレスを入力
3. 幅: 1920px、高さ: 1080px（推奨）
4. カスタムCSSは不要（テンプレートに含まれています）

## カスタマイズ

`style.css` の `:root` セクションで色をカスタマイズできます。

### グラデーションの色を変更
```css
:root {
    /* メイングラデーション */
    --g: linear-gradient(104deg, #53B7FF 0%, #81E3DD 94.77%);
}
```

例：ピンク系グラデーションに変更
```css
--g: linear-gradient(104deg, #FF53B7 0%, #DDA1E3 94.77%);
```

### 通常コメントの色を変更
```css
:root {
    /* 名前タグ */
    --listener-name: #FFFFFF;           /* 名前の文字色 */
    --listener-name-bg: var(--g);       /* 名前の背景色 */

    /* メッセージ本文 */
    --listener-comment: #333333;        /* 本文の文字色 */
    --listener-comment-bg: #FFFFFF;     /* 本文の背景色 */
}
```

### スパチャ/メンバーシップの色を変更
```css
:root {
    --superchat-bg: var(--g);           /* 背景色 */
    --superchat-comment: #FFFFFF;       /* 文字色 */
}
```
